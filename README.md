## Beschrijving
Ontwerpen en bouwen van een data-driven, interactieve detailpagina voor de Grote Elektriseermachine met Leidse flessen. De pagina moet het verhaal van dit object op een toegankelijke, aantrekkelijke en educatieve manier vertellen aan kinderen uit de bovenbouw van de basisschool.

Deze pagina dient als een proof of concept voor een groter digitaal museumplatform waarin verhalen, objecten en educatieve werkvormen als herbruikbare componenten worden aangeboden. Het uiteindelijke doel is om musea met een beperkt budget toegang te geven tot kant-en-klare componenten voor een moderne museumwebsite.

## Installatie

1. Ga naar [nodejs.org](https://nodejs.org) en installeer **Node.js 24.13.0 LTS** (Long Term Support).
2. Fork de repository en open het project in VSCodium.
3. Open de terminal in VSCodium en voer het volgende commando uit:
```bash
   npm install
```
4. Start het project met:
```bash
   npm start
```
   Het project is nu bereikbaar op **http://localhost:8000**.
5. Wanneer je klaar bent, stop je de server met **Control + C** in de terminal.
